# Lesson 2 — Strings (Summary)

Short summary of Python concepts covered in this session (Email Analyzer demo).

- Topic: string splitting and indexing (email -> username, domain)
- str.split() to break strings
- Accessing list elements by index (parts[0], parts[1])
- Basic input validation (check for '@' and non-empty parts)
- Formatting output for display (print or return formatted string)
- Small reusable function: analyze_email(email)

Extras covered in the project
- Running Python in the browser with Pyodide (no server required)
- JavaScript fallback for environments without Pyodide
- Simple front-end wiring: input, run button, output area
- Deployment note: static hosting via GitHub Pages
